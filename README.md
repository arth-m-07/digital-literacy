## Project Overview
This repository contains my portfolio for the Digital Literacy (CSE0001) course. It demonstrates practical skills in online safety, professional communication, and digital presence management.

## Module Summaries
* **Task 1: Digital Awareness:** An infographic highlighting safe internet practices and professional online presence.
* **Task 2: Digital Portfolio:** Screenshots establishing a foundational professional presence on GitHub and LinkedIn.
* **Task 3: Platforms:** Proof of completing Python coding challenges on HackerRank and a Google Form literacy quiz.
* **Task 4: Email Etiquette:** Drafts of professional academic and internship emails, plus a social media safety checklist.
* **Task 5: Cybercrime:** A case study on UPI payment fraud and an actionable prevention checklist.
